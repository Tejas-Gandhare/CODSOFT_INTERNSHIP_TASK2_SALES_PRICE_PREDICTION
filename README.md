# 📈 Sales Prediction using Python

## 📌 Overview
This project focuses on predicting product sales based on advertising expenditures across different platforms such as 📺 TV, 📻 Radio, and 📰 Newspaper. Machine Learning techniques are used to analyze advertising data and forecast future sales accurately.

This project was developed as **Task 2** of the :contentReference[oaicite:0]{index=0} Internship.

- 🐍 Python
- 🐼 Pandas
- 🔢 NumPy
- 📊 Matplotlib
- 📉 Seaborn
- 🤖 Scikit-learn
- 💾 Pickle


## 📂 Dataset
The dataset contains advertising expenditures on:

- Sales
